# Hello world

This is the basic print used in Orange Cat.

## Syntax

```ocat
print ( -string- )
```

Use `print` to print a string.
Between the parentheses, you can write any valid OCAT string.

### Params

- string: The string to be printed.

## Example

```ocat
print ( "Hello world!" )
```

## Notes

> [!IMPORTANT]
> Use the space character between the tokens.
