# Relative Path

You can use relative paths to load files.

## Syntax

```json
{
    "@/": "path/to/your/folder"
}
```

Use `{"@/": "path/to/your/folder"}` to define the relative path.

### Params

- path: The path to the folder.

## Example

```json
{
    "@/": "./src/"
}
```

## Notes

> [!IMPORTANT]
> Use the space character between the tokens.
